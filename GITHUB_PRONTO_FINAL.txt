                    📝 COMO COMEÇAR NO GITHUB

1. Criar repositório no GitHub

   git branch -M main
   git push -u origin main

3. Pronto! ✨


════════════════════════════════════════════════════════════════════════════════
                    🚀 COMO CLONAR E RODAR
════════════════════════════════════════════════════════════════════════════════

Alguém que clonar seu repositório vai fazer:

1. git clone https://github.com/seu-usuario/literalura.git
   cd literalura

2. cp .env.example .env

3. Editar .env com suas credenciais PostgreSQL

4. Criar banco de dados:
   psql -U postgres -c "CREATE DATABASE literatura;"

5. Compilar e rodar:
   mvn clean install
   mvn spring-boot:run
