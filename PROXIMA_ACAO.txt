SOLUÇÃO 1: Resetar PostgreSQL Service (2 minutos)
────────────────────────────────────────────────

1. Pressione: Windows + R
2. Digite: services.msc
3. Procure: postgresql-x64-15
4. Clique direito → Properties
5. Na aba "Log On": selecione "Local System account"
6. Clique "Apply" → "OK"
7. Clique direito no serviço → "Restart"
8. Tente conectar novamente


SOLUÇÃO 2: Reinstalar PostgreSQL (5 minutos)
──────────────────────────────────────────────

1. Desinstale PostgreSQL
2. Baixe novamente: https://www.postgresql.org/download/windows/
3. Durante instalação:
   - Database Superuser Password: postgres
   - Port: 5432
4. Termine instalação
5. Tente conectar novamente


SOLUÇÃO 3: Usar Senha Correta
──────────────────────────────

1. Descubra sua senha do PostgreSQL
2. Edite: src/main/resources/application.properties
3. Mude: spring.datasource.password=SUASENHA
4. Tente conectar novamente
