# LiterAlura

Um aplicativo CLI para descobrir e organizar livros da API Gutendex.
Busque livros de escritores famosos, organize por idioma e veja quais
autores estavam vivos em determinadas épocas.

## O que é?

LiterAlura é uma aplicação de linha de comando que conecta à API Gutendex
para buscar livros de domínio público. Você pode salvar seus livros favoritos
em um banco de dados PostgreSQL, filtrar por idioma e explorar informações
sobre os autores.

## Tecnologias

- Java 17+
- Spring Boot 3.3.0
- Spring Data JPA
- Hibernate ORM
- PostgreSQL
- Maven

## Como rodar localmente

### Pré-requisitos
- Java 17+
- Maven 3.6+
- PostgreSQL 12+

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/literalura.git
cd literalura
```

2. Configure as variáveis de ambiente:
```bash
cp .env.example .env
```

3. Edite o arquivo `.env` com suas credenciais:
```ini
DB_URL=jdbc:postgresql://localhost:5432/literatura
DB_USER=postgres
DB_PASSWORD=sua_senha_aqui
```

4. Crie o banco de dados:
```bash
psql -U postgres -c "CREATE DATABASE literatura;"
```

5. Compile e execute:
```bash
mvn clean install
mvn spring-boot:run
```

## Funcionalidades

1. **Buscar livro pelo título** - Busca na API Gutendex
2. **Listar livros registrados** - Mostra livros salvos
3. **Listar autores** - Lista autores com seus livros
4. **Listar autores vivos em ano** - Filtra por ano específico
5. **Listar livros por idioma** - Filtra por idioma

## Variáveis de Ambiente

| Variável | Descrição | Exemplo |
|----------|-----------|---------|
| `DB_URL` | URL PostgreSQL | `jdbc:postgresql://localhost:5432/literatura` |
| `DB_USER` | Usuário | `postgres` |
| `DB_PASSWORD` | Senha | `sua_senha_secreta` |
